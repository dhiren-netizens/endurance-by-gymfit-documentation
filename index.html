Installation

This document has been written in a way that is easily understood by both beginners and advanced users, including developers.

Follow the steps below to set up the theme:

  1. Installation via WordPress Admin area
    • Login into your WordPress admin area.
    • Go to Dashboard » Appearance » Themes.

    • Now, Click on Upload Theme button at top and upload endurance-by-gymfit.zip which you have downloaded from wordpress.org.

    • Click on Activate once upload has finished.

  2. Installation Via FTP
    • Download endurance-by-gymfit file from wordpress.org.
    • Extract ZIP file
    • Using your FTP program, upload the non-zipped theme folder into the /wp-content/themes/ folder on your server.
    • Go to Dashboard » Appearance » Themes and find Endurance theme from list.
    • Click on Activate to active Endurance theme.

Install & Activate Plugins

After the Endurance theme is activated, a notification appears at the top suggesting the plugins required by the theme.

Required Plugins:
  • Contact Form 7: Manage multiple contact forms, plus you can customize the form and the mail contents flexibly with simple markup.
  • Wordfence: Monitor, detect, and block threats in real-time on your WordPress site, giving you complete control over your security.

Quick Guide

  1. Login into your WordPress admin area.
  2. Go to Appearance » Themes.
  3. Install the theme by searching for the theme name Endurance. Click on Install Now, then activate the theme. For detailed installation guide, please refer to the Installation section.
  4. After activation, a menu will appear in the dashboard for theme options.
  5. Now go to Settings » Reading
  6. There are multiple settings available for the homepage and blog page.
    • Click on the Static Page option under Your homepage displays.
    • This will enable the 'Homepage' and 'Posts page' dropdowns. Select Home for Homepage & Blogs for Posts Page
    • Then, click on Save Changes
  7. Now go to Settings » Permalinks

  8. Select the Post Name option in the Permalink Structure and click on Save Changes.

  9. Now open the frontend of the website; you will see that the correct homepage is displayed.
Congratulations! You have successfully configured the Endurance Theme.

Translation

Endurance Theme is language compatible, thus can be translated into multiple languages. You can use Poedit plugin for translation.

  1. Install Poedit tool and open it.
  2. Open the .POT file located at: /wp-content/themes/endurance-by-gymfit/languages/
  3. Click on Save.
  4. When prompted, enter your language code (e.g., en_GB for English - United Kingdom).
  5. Then click on Save to save your translated file.
  6. Save the file using the format: endurance-by-gymfit-en_GB.po for English (United Kingdom) language.
  7. Click on Save button.
  8. Save the file inside the languages folder of the theme.
  9. Select any string you want to translate. At the bottom, you’ll find two sections:
    • Source Text – keep this unchanged.
    • Translation – enter your translated version of the selected string.
  10. Source Text keep as it is and in Translation section enter your translated string of selected string that you want to translate and then save it.
  11. Repeat step 10 untill your all strings to be translate.
  12. Poedit will generate two files: a .po file and a .mo file.
    13. Note: When you save the files, you must name them according to your language code. Find a list of language codes at WordPress in your Language. For example, the language code for English (United Kingdom) is en_GB, so you would save the translated files as en_GB.po and en_GB.mo.
  13. Open your wp-config.php file and add the following line: define('WPLANG', 'Your language code'); for Ex.(en_GB for English (United Kingdom)) then save it.
  14. From the WordPress admin panel, go to: Dashboard » Settings » General and select the appropriate Site Language.
  15. Visit the frontend of your website and check if the language prefix appears correctly in the HTML and content.
For more about translating WordPress themes please visit the following link : https://developer.wordpress.org/themes/functionality/localization/
